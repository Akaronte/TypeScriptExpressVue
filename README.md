# TypeScriptExpressVue
Example a simple Server TypeScript Node Express Vue

#1 Install Dependencies
```
npm install
```
#2 Compile Webpack
```
npm run build
```

#3 Compile Node Express
```
npm run tsc
```

#4 Start Server
```
npm run serve
```

Now localhost port 3000 show Vue and localhost:3000/welcome for express functions
